### Enum

**enum**은 공통된 주제에 대해서 이미 **정해놓은 값** 만 받고 싶을 때 사용해요! 

사용 방법 : 
```swift
enum major{
    case iOS
    case Android
    case Frontend
    case Backend
}

```
위와 같이 열거형을 정의할 수 있어요!
또는
```swift
enum major{
    case iOS, Android, Frontend, Backend
}

```
위와 같이 적어줄 수도 있어요!
